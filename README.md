# CRM navigator automation

This project is automating a CRM navigator system with test using webdriverio framework

## Installation

```bash
git clone https://github.com/Umut-Arpidinov/QA_CRM.git
```
## Runing a project
Navigate to folder
```bash
cd QR_CRM
```
Install dependencies
```bash
npm install
```
Run project
```bash
npm run wdio
```
